# Магазин обуви

## Описание проекта
Данный проект представляет собой информационную систему для автоматизации работы магазина обуви. Система позволяет управлять каталогом товаров, обрабатывать заказы и разграничивать права доступа для различных ролей пользователей (Администратор, Менеджер, Клиент, Гость).

Проект является практической работой №1 по дисциприне методология разработки программного обеспечения.

## Технологический стек
*   Backend: Python, Flask, SQLAlchemy, Pydantic, JWT, Pylint.
*   Frontend: React, Axios, React Router Dom, PropTypes.
*   База данных: PostgreSQL.
*   Архитектура: REST API + Client-Server.

## Структура проекта
```text
/shoe_store_project
|-- /backend              # Исходный код API (Python/Flask)
|   |-- app.py            # Точка входа, эндпоинты
|   |-- models.py         # SQLAlchemy модели (ORM)
|   |-- schemas.py        # Pydantic схемы (Validation)
|   |-- db.py             # Конфигурация БД
|   |-- .env              # Переменные окружения (секреты)
|
|-- /client               # Исходный код интерфейса (React)
|   |-- /src
|   |   |-- /components   # Переиспользуемые компоненты (ProductCard, OrderCard)
|   |   |-- /pages        # Страницы (Login, Products, Form)
|   |   |-- App.jsx       # Роутинг (React Router)
```
